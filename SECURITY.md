# Security Policy

## Reporting a Vulnerability

If you discover a potential security vulnerability within this repository, please report it responsibly:

1. **Email the Maintainers**: Send an email to `vedh.mitmpl2024@learner.manipal.edu` detailing the vulnerability.
2. **Details to Include**:
   - Step-by-step instructions to reproduce the issue.
   - Potential impact of the exploit.
   - Any suggested mitigations.
3. **Response Timeline**: We aim to acknowledge reports within 48 hours and provide a fix or public disclosure timeline within 7 days.

Do not open a public GitHub issue for security exploits or vulnerabilities.
